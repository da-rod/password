# genpwd

## Installation

```bash
$ go install github.com/da-rod/password/cmd/genpwd
```

## Usage

```bash
$ genpwd -h
Usage of genpwd:
  -d	Include digits (default true)
  -l int
    	Length of the generated passwords (default 25)
  -low
    	Include lower case characters (default true)
  -n int
    	Number of passwords to generate (default 25)
  -s	Include symbols (default true)
  -up
    	Include upper case characters (default true)
```
